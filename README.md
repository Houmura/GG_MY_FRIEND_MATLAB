# GG_MY_FRIEND_MATLAB

# 混合卡池抽卡沉船计算器

 运行环境：MATLAB
 
 运行前请根据实际修改参数
 
 N 为抽取次数
 
 a, b, c, d, e 分别对应五种卡片的抽取概率(单位：%) 
 
 P为抽取N卡池，至少有一种卡片没被抽出(沉船)的概率(单位：%)
 
 1-P% 就是毕业的概率啦~！(单位：%)
 
 支持最多5种卡片的抽卡沉船计算
 
 * 单一卡池，将概率填入a，输出结果为P1
 * 双卡卡池，将概率填入a和 b，输出结果为P2
 * 三卡卡池，将概率填入a，b和 c，输出结果为P3
 * 以此类推
